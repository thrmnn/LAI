\caption{\textbf{End-to-end methodology.}
The framework comprises two stages.
The \emph{core pipeline} (top) detects individual trees from aerial imagery
using deep-learning crown segmentation and estimates per-tree LAI from
LiDAR point clouds via voxelization and Beer--Lambert light-extinction
modelling.
The \emph{extensibility layer} (bottom) uses the LiDAR-derived LAI values
as training targets for a gradient-boosted regression model, enabling
LAI prediction from tree morphology and genus alone in areas where LiDAR
data are unavailable.}
\label{fig:pipeline}
