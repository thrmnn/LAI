\caption{\textbf{End-to-end methodology.} The framework comprises two stages. The \emph{core pipeline} (top) detects individual trees from aerial imagery using deep learning and estimates per-tree LAI from LiDAR point clouds via voxelization and Beer--Lambert light extinction modelling. The \emph{extensibility layer} (bottom) trains a gradient-boosted regression model on the LiDAR-derived LAI values, enabling LAI prediction from tree morphology and genus alone in areas where LiDAR data are unavailable. Dashed arrow indicates optional LiDAR input for CHM generation.}
