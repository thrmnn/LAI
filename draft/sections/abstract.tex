Urban Heat Islands, intensified by climate change, raise city temperatures compared to surrounding rural areas. Although urban trees offer cooling benefits, traditional inventories lack detailed shade assessments across both public and private land. This study presents an automated framework for tree inventories with canopy density estimaion using  Amsterdam as a case study. We integrate high-resolution aerial imagery with NIR bands (0.25m resolution) using zero-shot deep learning for crown segmentation, and LiDAR point clouds (25 points/m²) for height and Leaf Area Index (LAI) estimation. LAI is derived through voxel-based processing (0.5m³ resolution) using leaf-on measurements. With data from 12,350 trees, our machine learning regression model predicts LAI from morphological parameters and genus, achieving 78.4\% accuracy compared to ground-truth data. This approach enables city-wide canopy analysis that goes beyond property boundaries, providing critical data for evidence-based policy development. The methodology reduces inventory time while offering scalability for data-sparse regions, supporting urban planners in developing targeted macro-level interventions such as land use requirements and tree regulations that can influence both public and private tree coverage to mitigate urban heat vulnerability and enhance climate resilience.
